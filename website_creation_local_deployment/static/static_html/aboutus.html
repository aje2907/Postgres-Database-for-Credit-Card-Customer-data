About Us



This is a Banking application that was created as part of the Final Project for the course CSE 4/560 - Data Models and Query Language.


This course is a part of core requirement for Engineering Science - Data Science program for Spring 2023.


The developers of this project are current graduate students at the University at Buffalo.


This application connects to the Postgres Database Milestone1_db on local. This application is deployed on local currently.


The website is functional and supports Read, Delete, Update and Insert queries.


At this time, the website may not include all the functionalities that are possible in conventional SQL. However, they may be included in future work.


We have followed all Academic Integrity rules and regulations as per the course and university requirements.
